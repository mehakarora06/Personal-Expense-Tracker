# Personal Expense Tracker

## Overview

Personal Expense Tracker is a simple Python application that helps users record and manage their daily expenses. The program allows users to add expenses, view all expense records, and generate spending summaries by category. Expense data is stored in a JSON file, ensuring that records are saved even after the program is closed.

## Features

* Add new expenses with amount, category, and date.
* View all recorded expenses.
* Display total spending.
* Display category-wise spending summary.
* Save expense records using JSON file handling.
* Load previously saved expense data automatically.

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Run the following command:

python exp_tracker.py
